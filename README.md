# SQL Business Analysis – Facturación, Ventas, Pagos y Productos

Este repositorio contiene una colección de consultas SQL reales utilizadas en análisis de negocio para:

- Facturación
- Pagos y conciliación
- Ventas por trimestre
- Devoluciones
- Precios por sucursal
- Análisis neto de ventas
- Reportes operativos por agente

Las consultas provienen de casos reales en entornos de distribución y ventas, adaptadas para uso educativo sin dependencias a una base de datos específica.

Todas están organizadas por categoría y cuentan con comentarios y formato profesional.
